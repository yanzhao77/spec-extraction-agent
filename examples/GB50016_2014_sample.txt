《建筑设计防火规范》GB 50016-2014（摘录）

第3章 建筑分类和耐火等级

3.1 建筑分类
3.1.1 建筑应根据其使用功能、火灾危险性、疏散难度和扑救难度等因素进行分类。
3.1.2 建筑分为以下几类：
    a) 甲类：生产火灾危险性最大的物品的厂房和仓库；
    b) 乙类：生产火灾危险性较大的物品的厂房和仓库；
    c) 丙类：生产火灾危险性中等的物品的厂房和仓库；
    d) 丁类：生产火灾危险性较小的物品的厂房和仓库；
    e) 戊类：生产火灾危险性最小的物品的厂房和仓库。

3.2 建筑耐火等级
3.2.1 建筑的耐火等级应分为一级、二级、三级和四级。
3.2.2 建筑耐火等级应根据建筑的类别、高度、使用功能等因素确定。
3.2.3 不同耐火等级建筑的主要构件燃烧性能和耐火极限应符合表3.2.3的规定。

第5章 总平面布局

5.1 一般规定
5.1.1 建筑的防火间距应根据建筑的火灾危险性、建筑高度、所在地区的消防力量等因素确定。
5.1.2 相邻两座建筑之间的防火间距不应小于表5.1.2规定的数值。

表5.1.2 相邻建筑的最小防火间距
┌──────────────────┬──────────┬──────────┬──────────┐
│ 建筑类别组合      │ 防火间距 │ 条件     │ 备注     │
├──────────────────┼──────────┼──────────┼──────────┤
│ 甲类与其他建筑    │ 50m      │ 必须     │ 无例外   │
│ 乙类与丙类        │ 13m      │ 正常情况 │ -        │
│ 乙类与丁类        │ 9m       │ 正常情况 │ -        │
│ 丙类与丁类        │ 6m       │ 正常情况 │ -        │
└──────────────────┴──────────┴──────────┴──────────┘

5.2 防火间距的计算
5.2.1 防火间距应按相邻建筑外墙面之间的最近距离计算。
5.2.2 当相邻建筑中有一座建筑的外墙为防火墙时，防火间距可减少50%，但不应小于表5.1.2规定距离的一半。
5.2.3 对于高度不超过24m的建筑，防火间距可按表5.1.2的规定减少25%。

第7章 防火分隔

7.1 防火墙
7.1.1 防火墙应采用不燃烧体材料制作，其耐火极限不应低于4.00h。
7.1.2 防火墙应砌至屋顶，并应超出屋面不小于0.5m。
7.1.3 防火墙上不应开设门、窗、洞口，但必须开设时，应采用甲级防火门、窗。

7.2 防火门
7.2.1 防火门应采用不燃烧体材料或难燃烧体材料制作。
7.2.2 防火门的耐火极限应符合下列规定：
    a) 甲级防火门：不应低于1.50h；
    b) 乙级防火门：不应低于1.00h；
    c) 丙级防火门：不应低于0.50h。
7.2.3 防火门应具有自闭功能，并应配备防火门闭门器。

第9章 建筑材料

9.1 一般规定
9.1.1 建筑内部装修材料的燃烧性能等级应分为A级、B1级、B2级、B3级。
9.1.2 建筑内部装修应采用不燃烧材料或难燃烧材料。

9.2 材料燃烧性能等级
9.2.1 A级：不燃烧体。
9.2.2 B1级：难燃烧体。
9.2.3 B2级：可燃烧体。
9.2.4 B3级：易燃烧体。

9.3 不同场所的材料选择
9.3.1 疏散走道、楼梯间、前室等人员密集场所的内部装修材料应采用A级不燃烧材料。
9.3.2 其他场所可采用B1级难燃烧材料，但不应采用B3级易燃烧材料。
9.3.3 对于高度超过100m的建筑，所有内部装修材料应采用A级不燃烧材料。

第11章 安全疏散

11.1 一般规定
11.1.1 建筑应设置两个或两个以上的安全出口，并应分散布置。
11.1.2 建筑的疏散楼梯间应采用防烟楼梯间或封闭楼梯间。

11.2 疏散楼梯间
11.2.1 防烟楼梯间应设前室，前室面积不应小于6.0m²。
11.2.2 疏散楼梯间的最小净宽度不应小于1.10m。
11.2.3 疏散楼梯间的耐火极限应符合表3.2.3的规定。

11.3 安全出口
11.3.1 安全出口的宽度不应小于0.90m，高度不应小于2.10m。
11.3.2 安全出口应直接通向室外或通向安全区域。
